# دانلودر فایل با اینترنت ملی
با این مخزن میتونید با استفاده از actions ها گیت هاب فایل هایی که نیاز دارید و دسترسی به آن با اینترنت ملی دسترسی ندارید رو دانلود کنید. 
- دانلود فایل با استفاده از لینک دانلود و ذخیره در مخزن
- دانلود یک صفحه کامل وبسایت
- دانلود ویدیو از یوتیوب

اگر لینک دانلود فایلی را ندارید میتوانید ادرس صفحه وب  سایت را از گوگل دریافت کنید و از اکشن browser ان صفحه را ذخیره کنید تا لینک های آن صفحه و اسکرین شاتش در اختیار شما قرار بگیرد.

کافی هست فقط این مخزن را fork کنید و در بخش actions ها نوع اکشن که میخواهید را اجرا کنید و مقادیری ضروری مثل لینک را وارد کنید . 
بعد در مخزن فایل ذخیره می شود و میتوانید فایل را دانلود کنید یا مخزن را pull کنید.

با تشکر از  [نیکزاد آواسام](https://github.com/nikzad-avasam/downloader) نویسنده اصلی این اکشن ها. این مخزن کپی شده و مقداری شخصی سازی شده است.


## تغییراتی که انجام شده
مقداری تغییر در کد ها انجام شده که به شرح زیر است:
- مشکل حجم زیاد مخزن بعد از استفاده 
- حذف تاریخچه کل  دانلود ها از کامیت ها با اکشن cleaner
